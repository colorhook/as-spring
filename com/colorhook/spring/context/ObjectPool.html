Packagecom.colorhook.spring.context
Classpublic class ObjectPool
InheritanceObjectPool Object

ObjectPool is used in class ContextInfo to cache the bean which is a singleton. ObjectPool类是一个对象池，用来保存对象的引用。 在配置文件中，如果一个Bean被指定为singleton，那么这个Bean在创建后会注册到ObjectPool中。 使用ContextInfo.getBean(id)来创建一个bean时，会首先查找ObjectPool中是否包含改Bean，如果是则返回ObjectPool中 的Bean，否则会重新生成一个Bean。
